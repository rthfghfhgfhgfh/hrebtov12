**Автор:** [Хребтов]

## Описание

Приложение «Random Quote Generator» — это GUI‑программа на Python, которая генерирует случайные цитаты из предопределённого списка. Программа позволяет:
* генерировать случайные цитаты;
* фильтровать цитаты по автору и теме;
* просматривать историю сгенерированных цитат;
* сохранять и загружать данные в формате JSON.

## Функциональность

* **Генерация цитат:** кнопка «Сгенерировать цитату» выбирает случайную цитату из списка.
* **История:** отображение последних 10 сгенерированных цитат.
* **Фильтрация:** возможность фильтровать цитаты по автору или теме.
* **Сохранение данных:** история и список цитат сохраняются в файл `quotes_data.json`.
* **Валидация:** проверка на пустые строки при добавлении новых цитат (в текущей реализации — базовый функционал, можно расширить).
